## Design Philosophy: The Apple Standard

### Core Principles (Non-Negotiable)

**1. Minimalism & Clarity**
- Remove everything that doesn't serve a purpose
- Every element must earn its place on the page
- White space is not empty space - it's a design element
- Focus user attention on what matters most

**2. Visual Hierarchy**
- Large, bold headlines that command attention (like Apple's product launches)
- Clear typographic scale (hero text → subheadings → body → captions)
- Strategic use of size, weight, and color to guide the eye
- One primary call-to-action per section

**3. Premium Typography**
- Use system fonts or high-quality web fonts
- Large, readable font sizes (minimum 16px for body text)
- Generous line-height (1.5-1.8) for readability
- Limited font weights (2-3 maximum)
- Ample letter-spacing for headlines

**4. Sophisticated Color Palette**
- Neutral base colors (whites, grays, blacks)
- One or two accent colors used sparingly
- High contrast for readability
- Subtle gradients (if used at all)
- Dark mode consideration

**5. High-Quality Imagery**
- Large, high-resolution images (but optimized for web)
- Professional photography or illustrations
- Images that tell a story
- Consistent visual style across all images
- Proper aspect ratios and cropping

**6. Smooth, Purposeful Animation**
- Subtle fade-ins and slide-ups on scroll
- Smooth parallax effects (used sparingly)
- Hover states that feel responsive
- Page transitions that feel natural
- 60fps performance minimum
- Animations that enhance understanding, not just decoration

**7. Generous Spacing**
- Large padding and margins (think "breathing room")
- Consistent spacing scale (8px, 16px, 24px, 32px, 48px, 64px, 96px, 128px)
- Vertical rhythm that creates flow
- Sections clearly separated

**8. Perfect Responsiveness**
- Mobile-first approach
- Seamless experience across all devices
- Touch-friendly targets (minimum 44x44px)
- Optimized images for different screen sizes
- Consistent experience, not identical

**9. Performance Excellence**
- Fast initial load (< 2 seconds)
- Smooth scrolling and interactions
- Optimized images (WebP, lazy loading)
- Minimal JavaScript bundle size
- Progressive enhancement

**10. Accessibility First**
- WCAG 2.1 AA minimum (aim for AAA)
- Keyboard navigation perfection
- Screen reader friendly
- Sufficient color contrast
- Focus indicators that are beautiful AND functional
