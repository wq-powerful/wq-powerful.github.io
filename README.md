- 💡 **产品思维** - 把零散经验压缩成清晰的产品表达

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 部署到 GitHub Pages
npm run build && npx gh-pages -d dist -b gh-pages
```
