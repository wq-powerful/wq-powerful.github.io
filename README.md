## 项目简介

这是一份使用 AI 辅助构建的个人品牌门户网站，作为 Inspire（原 Thoughtworks 中国）2026 校招 AI 产品经理岗位作业的回答。

网站通过自然语言对话驱动 AI 代码助手（VS Code + Claude Code）完成从 0 到 1 的构建，展示了我在业务理解、需求转化、AI 工具运用与落地执行方面的综合能力。

## 核心考察点回应

### 1. 拥抱 AI 辅助（AI-Assisted）

- 全程使用 AI 代码助手（VS Code + Claude Code）进行页面框架搭建、组件开发、样式调优
- 通过精准的 Prompt 定义产品结构、交互逻辑与视觉风格
- 网站中专设「我如何借助 AI 完成 HomeWork」板块，透明展示 AI 协作的五步流程
- AI 使用原则：AI 是加速器，不是替代品——最终判断、内容筛选和体验优化均由本人完成

### 2. 产品思维与审美

- 信息架构清晰：首页定位 → 关于 → 优势 → 项目 → 方法论 → 赋能 → AI 实践 → 联系
- 视觉设计借鉴 Apple 极简风格：大量留白、精致排版、微妙动画、高级感配色
- 每个板块占满一屏，滚动体验流畅，内容层次分明
- 交互细节：毛玻璃导航栏、卡片 hover 上浮、标签页切换动画、数据指标入场动画

### 3. 快速迭代能力

- 从 MVP 骨架到完整功能经历多轮迭代优化
- 迭代路径：基础框架 → 内容填充 → 视觉升级 → 板块重构 → 间距调优 → 部署上线
- 使用 GitHub Actions 实现自动构建部署，推送即上线
